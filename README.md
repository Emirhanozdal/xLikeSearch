# X Like Search

Chrome extension for turning liked tweets into a local searchable archive with a product-grade side panel.

## Current Product Core

- MV3 Chrome extension with side panel workflow
- Local-first archive using `chrome.storage.local`
- Active tab awareness to validate the X likes page before sync
- Search, topic filters, quick prompts, and result cards
- Heuristic topic tagging for `rag`, `fine-tuning`, `agents`, `evals`, `infra`, `product`, and `design`
- Product-style shadcn-inspired UI with iconography and dashboard stats

## Planned Retrieval Upgrades

- Auto-scroll archive import for deeper sync coverage
- Embeddings-based semantic search
- Hybrid retrieval: keyword + semantic + metadata
- "Ask my likes" answer mode on top of retrieval
